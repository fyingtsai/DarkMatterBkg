# DarkMatterBkg
subjStudy.cc Usage
-----
MC samples are listed in the file [Sample.h](Sample.h). 
Include Sample.h when you excute the file subjStudy.cc.
```ruby
.x subjStudy.cc+(id)
```
replace `id` by the number which corresponds to the contro region. See the table below. 

| id        | Contro Region     |
| ------------- |:-------------:|
| 1      | signal |
| 24     | w + jet      |
| 6      | ttbar     |
| 12     | z(nunu) + jet      |
